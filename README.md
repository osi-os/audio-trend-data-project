# audio-trend
Exploring content consumption patterns across music and podcasts on streaming platforms - 

Analyzing the audio ecosystem from three vantage points across 2022–2025. Podcast reviews capture historical listener sentiment (2022–2023), Spotify charts show music consumption trends (late 2023–2025), and podcast charts reveal spoken-word content trends (late 2024–2025). Where these datasets overlap — October 2024 to June 2025 — we can directly compare music and podcast consumption patterns.
